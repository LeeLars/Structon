# Industrie Afbeeldingen

Deze map bevat hero afbeeldingen voor de verschillende industrie pagina's.

## Benodigde Afbeeldingen

Upload de volgende afbeeldingen in deze map:

1. **grondwerkers.jpg** - Hero afbeelding voor Grondwerkers pagina
2. **afbraak-sloop.jpg** - Hero afbeelding voor Afbraak & Sloop pagina
3. **baggerwerken.jpg** - Hero afbeelding voor Baggerwerken pagina
4. **loonwerk-landbouw.jpg** - Hero afbeelding voor Loonwerk & Landbouw pagina
5. **recycling.jpg** - Hero afbeelding voor Recycling pagina
6. **tuinaanleggers.jpg** - Hero afbeelding voor Tuinaanleggers pagina
7. **verhuur.jpg** - Hero afbeelding voor Verhuur pagina
8. **wegenbouw.jpg** - Hero afbeelding voor Wegenbouw pagina

## Specificaties

- **Formaat**: JPG of PNG
- **Aanbevolen resolutie**: 1920x600px (minimaal 1600x500px)
- **Aspect ratio**: 16:5 of 3:1
- **Bestandsgrootte**: Max 500KB (geoptimaliseerd voor web)
- **Inhoud**: Relevante afbeelding die de industrie representeert (machines, werkzaamheden, etc.)

## Naamgeving

Gebruik exact de bestandsnamen zoals hierboven vermeld (lowercase, met streepjes).
De afbeeldingen worden automatisch geladen op basis van de industrie slug.
