3 KIES DE JUISTE BREEDTE

De breedte van je bak hangt af van het type werk:

Slotenbakken

  • 200-300mm: Kabelsleufjes, smalle leidingen
  • 400-500mm: Standaard leidingwerk, drainage
  • 600-800mm: Rioleringen, bredere sleuven
  • 1000-1200mm: Brede sloten, egalisatiewerk

Dieplepelbakken

  • 600-800mm: Compacte machines, preciezer werk
  • 900-1200mm: Standaard graafwerk
  • 1300-1500mm: Groot grondverzet, effici√´ntie

4 OVERWEEG EXTRA FEATURES

Afhankelijk van je werk kunnen extra features nuttig zijn:

  • Hardox slijtplaten: Voor abrasieve grond en intensief gebruik
  • Graaftanden: Voor harde grond en klei
  • Perforatie: Voor baggerwerk (water kan wegstromen)
  • Kantelmechanisme: Voor egalisatiewerk
